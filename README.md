MTG Manager

یک اسکریپت ساده و سریع برای نصب و مدیریت MTG v1 & MTG v2 روی سرور لینوکس.

## Install
```
bash <(curl -fsSL https://raw.githubusercontent.com/Ptechgithub/mtg_proxy_manager/main/install.sh)
```
![35](https://raw.githubusercontent.com/Ptechgithub/configs/main/media/35.jpg)

### امکانات:

- نصب MTG v1 و MTG v2

- پشتیبانی از AdTag (کانال اسپانسری)

- نمایش لینک پروکسی

- مدیریت سرویس‌ها (وضعیت، حذف، تغییر تگ)
